html2pdf.it
Convert your html to a pdf that doesn't suck.

By Allan Ebdrup

What is this?

This is a public API that has one purpose: Convert html from a given URL to a pdf that doesn't suck.

How does it work

You just call http://html2pdf.it/pdf?url=[url_to_convert_to_pdf]

Type in URL to generate a pdf

What are the features?

  • Selectable text in the pdf
  • Javascript on the page is run
  • Flash is not rendered

Why should I use this?

  • Converting html to pdf is a fast and easy way to create beautiful pdf files, you loose a little control over the exact layout. But you will be generating pdf files much faster than coding them by hand.
  • When running a website pdf generation is a CPU and memory-heavy operation. If you do the pdf-generation on your web servers and you get a surge in people requesting pdf files, your entire website might become slow. That's why you should generate pdf files on separate application-servers. Using html2pdf.it, we are those application-servers for you.

How stable is this service?

It's being run on several load balanced application servers with fail over, the up-time should be high.

Why is this free?

This is just an experiment to see if there is any interest in this kind of service. If we get a lot of interest there will be some kind of payment for the service, to cover server costs. If not, just use it for free and enjoy.
