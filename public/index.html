html2pdf.it

html2pdf that doesn't suck

html2pdf.it is a webservice, just pass a URL and get a PDF rendering.
All you need to do is call http://www.html2pdf.it/?url=[url_to_convert_to_pdf]

Open source

html2pdf.it is open source, take a look at the code, fork it, run your own html2pdt.it server.

Renders as a real browser

CSS is interpreted byt the WebKit rendering engine, also used in Chrome and Safari. JavaScript is run as if in a real browser.

Build with node.js

The technology used is node.js.

Nothing to install

If you do not want to install and run your own servers, you are welcome to use html2pdf.it directly here, by calling http://www.html2pdf.it/?url=[url_to_convert_to_pdf]. NOTE: if the traffic ramps up to cost me a lot of money, this may turn into a paid service, or require donations. Running your own server will always be free.
